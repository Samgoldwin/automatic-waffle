# Machine Learning Lab (BCSL606)

Machine Learning Laboratory programs implemented as part of the VTU 6th Semester Computer Science & Engineering curriculum.

## Overview

This repository contains implementations of fundamental Machine Learning algorithms and data analysis techniques using Python and Scikit-Learn. The experiments focus on data visualization, dimensionality reduction, supervised learning, unsupervised learning, regression, and classification techniques.

## Experiments Included

### 1. Exploratory Data Analysis (EDA)

* Histograms for numerical features
* Box plots for outlier detection
* IQR-based outlier identification
* Dataset: California Housing

### 2. Correlation Analysis

* Correlation matrix computation
* Heatmap visualization
* Pair plot generation
* Dataset: California Housing

### 3. Principal Component Analysis (PCA)

* Feature standardization
* Dimensionality reduction from 4D to 2D
* Visualization of principal components
* Dataset: Iris

### 4. Find-S Algorithm

* Concept learning
* Hypothesis generation
* Version space fundamentals
* Dataset: Custom CSV training data

### 5. k-Nearest Neighbors (KNN)

* Classification using different K values
* Performance comparison
* Accuracy evaluation
* Dataset: Randomly generated data

### 6. Locally Weighted Regression (LWR)

* Non-parametric regression
* Weighted least squares
* Mean Squared Error evaluation
* Dataset: California Housing

### 7. Linear & Polynomial Regression

* Linear Regression implementation
* Polynomial feature transformation
* Model comparison
* Housing and MPG prediction datasets

### 8. Decision Tree Classification

* Decision Tree construction
* Classification analysis
* Dataset: Breast Cancer Wisconsin Dataset

### 9. Naive Bayes Classification

* Gaussian Naive Bayes implementation
* Face recognition classification
* Dataset: Olivetti Faces Dataset

### 10. K-Means Clustering

* Unsupervised learning
* Cluster visualization
* Dataset: Breast Cancer Wisconsin Dataset
